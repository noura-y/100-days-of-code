# Apprentissage de Python - freeCodeCamp

Ce dépôt centralise ma progression, mes notes logiques et mes exercices pratiques réalisés dans le cadre de la certification "Scientific Computing with Python" de freeCodeCamp. L'objectif est de maîtriser la syntaxe et les structures algorithmiques fondamentales pour piloter efficacement des modèles d'intelligence artificielle et concevoir des scripts d'automatisation.

## Objectifs de formation
* Assimiler la logique algorithmique (boucles, conditions, fonctions).
* Structurer des scripts propres et réutilisables.
* Appliquer ces concepts à la résolution de problèmes techniques et d'ingénierie.
